# gulp-sample
frontend sample webserver

## Description
This webserver live build for Sass and ES6.

## Requirement
- Node.js

## Installation
```shell
$ git clone git@github.com:jaxx2104/node-sample.gi
$ npm run install
```
## Usage
live build start.
```shell
$ gulp
```

## Update
update node.js
```shell
$ sudo npm install -g n
$ sudo n latest
```

update npm global
```shell
$ sudo npm update -g npm
$ sudo npm update -g
$ sudo npm outdated -g
```

update npm local
```shell
$ sudo npm update
$ sudo npm outdated
```
